Getting Started
What to do on a fresh install before any data has been scraped
  1. Start the server
    Run uvicorn main:app --reload --port 8000 and open http://localhost:8000.
  2. Run your first scrape
    Go to Rate Intelligence. The dashboard will show mock/placeholder data until real data exists. Click the Scrape Now button (or change the date picker) to pull live prices from all 7 competitors. This takes about 30–60 seconds.
  3. Populate the Forward Rates horizon
    Go to Rate Intelligence → Forward Rates and click Scrape Horizon. This runs all 7 scrapers across the next 16 weekly pickup windows (112 scrape calls). Run this once manually — the daily scheduler will keep it fresh from 07:15 onwards.
  4. Populate Seasonal Analysis
    Go to Rate Intelligence → Seasonal Analysis and click Scrape All. This scrapes the 15th of each of the next 12 months (7-night stays). The Monday weekly cron job will update it automatically after that.
  5. Set up SEO tracking (optional)
    Go to Settings and enter your SerpAPI key. Then go to SEO Rank Tracker and click Check Rankings Now. 10 Iceland car rental keywords are pre-seeded — add or remove via the Keywords section in Settings.
  6. Configure Slack alerts (optional)
    Go to Settings → Slack Alerts and paste your webhook URL. The system will notify you automatically whenever a competitor undercuts Blue Car Rental's price on any model.
Automated Schedule
What runs automatically once the server is running
Job When What it does
Rate Scrape Daily 07:00 Live rates for today+7 → today+10, all 7 competitors
Horizon Scrape Daily 07:15 Next 16 weekly windows × 7 competitors (112 calls)
SEO Check Daily 07:30 Google rankings for all stored keywords via SerpAPI
Alert Check Daily 07:45 Fires Slack alert if any competitor undercuts Blue
Seasonal Scrape Every Monday 08:00 Re-scrapes all 12 monthly anchor dates (15th, 7-night stay)

The rate scrape, SEO check, and alert check schedule (daily / hourly / weekly) can be changed in Settings. The horizon and seasonal jobs always run on their fixed schedules.

What Each View Shows
Rate Intelligence tab breakdown
List View
All live rates in a sortable table. Change the date picker at the top to trigger a fresh scrape for any pickup/return window. The Δ column shows the price movement for each competitor since their previous scrape.
Car Model Matrix
Side-by-side price grid for every canonical car model across all competitors. Green = cheapest on that model, red = most expensive. If no data exists for the selected dates, it falls back to the most recent data available.
Price History
Time-series line charts showing how prices have evolved over the last 7, 14, 30, or 90 days. Useful for spotting trend patterns and seasonal drift.
Seasonal Analysis
Per-day pricing across the next 12 months (pickup on the 15th, 7-night stay). Switch to History mode to see how prices for a specific future month have changed across successive scrapes — the clearest signal for competitor yield management.
Forward Rates
Horizon chart and heatmap for the next 16 weekly pickup windows. Use the category filter pills to compare within a segment, or select a specific car model from the dropdown to see per-model competitor lines.
Tips
  • Unknown car models — go to Settings → Category Audit to see any models that haven't been mapped to a canonical name yet. You can add mappings via Settings → Car Model Mappings.
  • Mock data badge — if you see a "Mock Data" badge in the top bar, the DB has no real scraped data for the current filters. Run a scrape to populate it.
  • Insurance tab — click any price cell in the Price by Category table to edit it inline. Prices persist in the database and survive restarts. Use Trigger Research to open all competitor insurance pages at once, then Mark Reviewed when done.
  • Locations — Keflavik Airport and Reykjavik are the two Blue Car Rental locations. Lava and Lotus only operate from KEF. Add or rename locations via Settings if needed.
